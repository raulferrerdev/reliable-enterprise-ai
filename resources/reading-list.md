# Reading List

Curated references for building reliable enterprise RAG systems.

## Foundational papers

- **REALM** (Guu et al., 2020) — Retrieval-Augmented Language Model Pre-Training
- **RAG** (Lewis et al., 2020) — Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks
- **RAGAS** (Es et al., 2023) — Automated Evaluation of Retrieval Augmented Generation

## Retrieval

- **ColBERT** — Efficient and Effective Passage Search via Contextualized Late Interaction
- **HyDE** (Gao et al., 2022) — Precise Zero-Shot Dense Retrieval without Relevance Labels

## Reliability & evaluation

- **TruLens** documentation — LLM evaluation framework
- **ARES** — Automated RAG Evaluation System
