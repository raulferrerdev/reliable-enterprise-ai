# Contributing

This is a learning-in-public repository. Contributions, corrections and discussions are welcome.

## What kind of contributions

- **Bug fixes** in code examples
- **Corrections** to technical claims (with sources)
- **Alternative implementations** of the same pattern
- **Evaluation results** on different datasets or configurations

## What this is not

This is not a production library. It is a documented learning project. Code is written for clarity and learnability, not for direct production use without adaptation.

## Opening issues

If you find a technical error, open an issue with:
- The specific claim or code in question
- Why you think it is incorrect
- A reference or alternative if you have one

## Discussion

The most valuable contributions are often in the issues — sharing your own experience with these patterns in production, pointing out edge cases, or questioning assumptions.
